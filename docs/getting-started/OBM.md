# 出库管理系统用户手册

---

## 1. 功能概述

### 出库管理系统功能概述

出库管理系统是一款基于 JavaFX 和 MyBatis 的桌面应用程序，旨在为企业或仓储管理人员提供一个高效、直观且安全的库存出库管理平台。系统通过模块化设计，将各项功能拆分为独立的模块，既保证了系统的高内聚低耦合，又便于后期维护和扩展。以下是系统主要功能模块的概述：

---

### 功能模块

- **用户管理**
  - **登录与注册**  
    用户可以通过注册新账户、登录以及重置密码等功能进行账户管理。系统提供实时输入校验和详细的反馈信息，确保用户操作简单、安全。
  - **用户中心**  
    登录后，用户可以在个人中心查看和更新个人信息，或进行账户删除等操作，满足用户对账户数据管理的需求。

- **出库管理**
  - **出库操作**  
    系统支持记录和管理商品出库信息，实现库存数据的实时更新，帮助用户高效处理出库业务。
  - **数据查询与统计**  
    提供多条件查询和统计分析功能，便于用户对历史出库记录进行跟踪和数据分析，辅助决策。

- **数据持久化与映射**
  - **数据库操作**  
    基于 MyBatis 框架，通过 Mapper 接口与 XML 映射文件实现对用户和出库数据的增删改查，确保数据的持久存储与安全性。
  - **数据模型管理**  
    通过实体类与查询条件构造类（Example 类）的设计，清晰定义系统核心数据结构，支持复杂查询需求。

- **会话与状态管理**
  - **用户会话管理**  
    通过专门的会话管理模块维护用户登录状态和跨界面数据共享，确保整个应用程序的操作连续性和状态一致性。

- **辅助工具与安全验证**
  - **工具辅助模块**  
    包括文件操作、界面加载（FXML 加载）、图标管理等工具类，简化了开发流程并提高了代码复用性。
  - **安全验证**  
    采用密码加密、格式校验以及过期检查等多重安全措施，确保用户数据和操作的安全性。

- **用户界面**
  - **直观友好的 GUI**  
    利用 JavaFX 构建清晰简洁的图形用户界面，用户可以通过直观的布局快速找到所需功能，并通过实时反馈提升操作体验。
  - **响应式交互设计**  
    系统提供实时输入验证、错误提示和辅助说明，确保用户在使用过程中能够即时了解信息输入的正确性，减少操作失误。

---

通过以上各大功能模块的协同工作，出库管理系统实现了对库存出库业务全流程的全面支持，不仅提升了工作效率，还保障了数据安全和系统稳定性。无论是日常的出库操作还是复杂的业务数据统计分析，该系统都能为用户提供高效、可靠的解决方案。

---

## 2. 系统要求

- **操作系统**：支持 Windows、macOS 或 Linux（需安装 Java 环境）
- **Java 环境**：Java 17（包含与之对应的 JavaFX 运行环境）
- **应用程序**：已正确安装并启动出库管理系统桌面应用

## 3. 界面说明

### 登录界面

<img src="https://s2.loli.net/2025/03/22/qb1jL9eFv3YSUQW.png" alt="登录界面" style="width:50%;">

- **界面操作**
  - 程序启动进入登录界面，输入正确的账号密码，点击 **登录** 可进入出库管理系统的 **主界面**。
  - 如果尚未拥有账户，则可点击 **注册** 进入出库管理系统的 **注册界面**。
  - 若已拥有账户但忘记密码，则可点击 **忘记密码** 进入重置密码界面。
  - 点击 **退出系统** 可退出程序。

- **界面快捷操作**
  - 使用上下方向键可切换文本框。
  - 在账号框按回车键可切换至密码框。
  - 在密码框按回车键可触发登录操作。
  - 当账号和密码框均为空时，按回车键可触发注册操作。

### 注册界面

<img src="https://s2.loli.net/2025/03/22/RGdfQzxJ2TKm9Io.png" alt="注册界面" style="width:50%;">

- **界面操作**
  - 进入注册界面后，填写所有文本框并确保数据正确后，点击 **确认**。待系统验证数据合法后，即可注册新账户。
  - 注册成功后，系统会跳转回 **登录界面**，用户可使用注册的账号密码登录。
  - 注册失败时，系统会提示哪些字段不合法，用户需根据提示修改后重新注册。
  - 点击 **返回** 可返回到 **登录界面**。

- **界面快捷操作**
  - 使用上下方向键可切换文本框。
  - 每个文本框内按回车键可切换至下一个文本框。
  - 在最后一个文本框按回车键可触发确认操作。

### 重置密码界面

<img src="https://s2.loli.net/2025/03/22/tPGgJmhMQ6WKcZI.png" alt="重置密码界面" style="width:50%;">

- **界面操作**
  - 输入正确的账号、邮箱及手机号后，点击 **验证身份**。系统验证数据合法性后，允许用户通过安全问题重置密码。
  - 重置密码成功后，系统会跳转回 **登录界面**，用户可使用新密码登录系统。

- **界面快捷操作**
  - 使用上下方向键可切换文本框。
  - 每个文本框内按回车键可切换至下一个文本框。
  - 在手机号文本框按回车键可触发验证身份操作。
  - 在最后一个密码框按回车键可触发确认操作。

### 主界面

<img src="https://s2.loli.net/2025/03/23/y7lAGTkDMIHzaCU.png" alt="主界面" style="width:50%;">

- **界面操作**
  - **录入出库设备信息**
    - **默认模式**
      - 设备编号和设备 Mac 均不能为空，用户需输入完整信息。
      - 系统记录订单号、设备编号、设备 Mac 及设备 sn 码，并存入数据库。
      - 录入后，数据将加载至右侧表格。
    - **Id==Mac 模式**
      - 设备编号和设备 Mac 均不能为空，仅需输入设备编号。
      - 系统记录订单号、设备编号、设备 Mac 及设备 sn 码，并存入数据库。
      - 录入后，数据将加载至右侧表格。
    - **Just-Id 模式**
      - 仅需输入设备编号，设备 Mac 不可填写。
      - 系统记录订单号、设备编号及设备 sn 码，并存入数据库。
      - 录入后，数据将加载至右侧表格。
    - **Just-Mac 模式**
      - 仅需输入设备 Mac，设备编号不可填写。
      - 系统记录订单号及设备 Mac，并存入数据库。
      - 录入后，数据将加载至右侧表格。

  - **搜索出库设备信息**
    - 用户可在订单编号、设备编号或设备 Mac 的输入框中输入数据，点击 **查找** 按钮后，系统将在数据库中搜索匹配数据，并加载至右侧表格。
    - 搜索时，输入框内容可为空，但需完整匹配字段信息才能搜索成功。

  - **修改出库设备信息**
    - 在右侧表格中选中某条数据后，按回车键可将数据加载至左侧输入框。
    - 修改后点击 **修改** 按钮将数据保存至数据库。

  - **删除出库设备信息**
    - 在右侧表格中选中某条数据后，点击 **删除** 按钮将该数据从数据库中进行软删除。

- **界面快捷操作**
  - 使用上下方向键可切换文本框。
  - 每个文本框内按回车键可切换至下一个文本框。
  - 在最后一个文本框按回车键可触发录入操作。
